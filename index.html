What is COMOT?

Note: From here onwards we will refer to services and cloud applications with the same "services" term.

COMOT provides an easy to use integration layer for managing various aspects of elastic cloud services. In particular, COMOT provides a domain specific API to describe elastic cloud services, interracts with SALSA for deploying the description to virtualized infrastructures (e.g., OpenStack, Docker), relies on MELA for cloud service monitoring and analysis, and on rSYBL for run-time control of elastic cloud services.
