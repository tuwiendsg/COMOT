Guideline to run the COMOT demo on a single host.

This guide is going to introduce the step of running COMOT components on a single machine such as one VM or on user's PC.

  • Install and run COMOT dashboard
  • Create a HelloElasticity application from COMOT fluent API
  • Deploy the application by SALSA
  • Control the application by SYBL
  • Monitor the application by MELA
  • Analyze the elasticity by QUELE
